# Nacho Costero DJ - Web Oficial

Sitio web profesional de Nacho Costero DJ. Diseñado y desarrollado por Alejandro Lamas.

## Descripción

Web corporativa para Nacho Costero DJ, DJ profesional en Madrid especializado en bodas, eventos corporativos y fiestas privadas. El sitio incluye:

- Página principal con vídeo hero, galería y testimonios
- Sección de bodas con vídeo animado por scroll y cobertura musical completa
- Sección de eventos con galería dedicada
- Formulario de contacto funcional
- Páginas legales (aviso legal, política de privacidad, política de cookies)

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Lenguaje:** TypeScript
- **Estilos:** Tailwind CSS 4
- **Animaciones:** Framer Motion, GSAP (ScrollTrigger)
- **UI:** shadcn/ui
- **Email:** Nodemailer

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Build

```bash
npm run build
npm start
```

## Estructura del proyecto

```
src/
  app/          # Rutas y páginas (App Router)
  components/   # Componentes React
  lib/          # Utilidades y datos
public/
  images/       # Imágenes (bodas, eventos, personal)
  videos/       # Vídeos (bodas, general)
```
